# Product Context

## Problem Statement
Тренерам и игрокам водного поло необходим инструмент для визуализации и отработки тактических схем без необходимости проведения тренировок в бассейне.

## Target Users
- Тренеры водного поло
- Игроки водного поло (все уровни)
- Спортивные школы и секции

## User Needs
1. Возможность видеть тактические схемы в движении
2. Интерактивное управление расстановкой игроков
3. Переключение между различными тактическими сценариями
4. Понимание логики перемещений защитников при атакующих действиях

## Current Status
Проект на стадии инициализации. ТЗ определено, реализация не начата.
